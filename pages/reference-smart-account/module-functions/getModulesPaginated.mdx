# `getModulesPaginated`
Defined in [ModuleManager.sol](https://github.com/safe-global/safe-smart-account/blob/main/contracts/base/ModuleManager.sol)

## Overview

```solidity
function getModulesPaginated(address start, uint256 pageSize) external view returns (address[] array, address next)

```

Returns an array of modules. If all entries fit into a single page, the next pointer will be 0x1. If another page is present, next will be the last element of the returned array.

## Returns

### `array`
- **Type:** `address[]`

Array of modules.

### `next`
- **Type:** `address`

Start of the next page.

## Parameters

### `start`
- **Type:** `address`

Start of the page. Has to be a module or start pointer (0x1 address).

### `pageSize`
- **Type:** `uint256`

Maximum number of modules that should be returned. Has to be > 0.
