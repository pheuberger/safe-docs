# `disableModule`
Defined in [`ModuleManager.sol`](https://github.com/safe-global/safe-smart-account/blob/main/contracts/base/ModuleManager.sol#L142)

## Overview

```solidity
function disableModule(address prevModule, address module) external

```

Disables the module `module` for the Safe.

*This can only be done via a Safe transaction.*

This function emits the [`DisabledModule`](../module-events/DisabledModule.md) event.

## Parameters

### `prevModule`
- **Type:** `address`

Previous module in the modules linked list.

### `module`
- **Type:** `address`

Module to be removed.
