# `removeOwner`

## Overview

Removes the owner `owner` from the Safe and updates the threshold to `_threshold`.

<Callout type='warning'>This can only be done via a Safe transaction.</Callout>

## Parameters

### `prevOwner`

- **Type:** `address`

Owner that pointed to the owner to be removed in the linked list

```solidity focus=2
(ISafe safe).removeOwner(
    "0x...",
    "0x...",
    1
);
```

### `owner`

- **Type:** `address`

Owner address to be removed.

```solidity focus=3
(ISafe safe).removeOwner(
    "0x...",
    "0x...",
    1
);
```

### `_threshold`

- **Type:** `uint256`

New threshold.

```solidity focus=4
(ISafe safe).removeOwner(
    "0x...",
    "0x...",
    1
);
```

## Events

### `RemovedOwner`

```solidity
event RemovedOwner(address owner);
```

Emitted when an owner is removed from the Safe.

This event is also emitted in the [`swapOwner`](/reference-smart-account/owner/swapOwner.mdx) functions.
