# `addOwnerWithThreshold`
Defined in [`OwnerManager.sol`](https://github.com/safe-global/safe-smart-account/blob/main/contracts/base/OwnerManager.sol#L56)

## Overview

```solidity
function addOwnerWithThreshold(address owner, uint256 _threshold) external

```

Adds the owner `owner` to the Safe and updates the threshold to `_threshold`.

*This can only be done via a Safe transaction.*

This function emits the [`AddedOwner`](../owner-events/AddedOwner.md) and [`ChangedThreshold`](../owner-events/ChangedThreshold.md) events.

## Parameters

### `owner`
- **Type:** `address` 

New owner address.

### `_threshold`
- **Type:** `uint256`

New threshold.
