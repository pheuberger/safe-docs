# `swapOwner`
Defined in [`OwnerManager.sol`](https://github.com/safe-global/safe-smart-account/blob/main/contracts/base/OwnerManager.sol#L89)

## Overview

```solidity
function swapOwner(address prevOwner, address oldOwner, address newOwner) external

```

Replaces the owner `oldOwner` in the Safe with `newOwner`.

*This can only be done via a Safe transaction.*

This function emits the [`RemovedOwner`](../owner-events/RemovedOwner.md) and [`AddedOwner`](../owner-events/AddedOwner.md) events.

## Parameters
### `prevOwner`
- **Type:** `address`

Owner that pointed to the owner to be replaced in the linked list

### `oldOwner`
- **Type:** `address`

Owner address to be replaced.

### `newOwner`
- **Type:** `address`

New owner address.
