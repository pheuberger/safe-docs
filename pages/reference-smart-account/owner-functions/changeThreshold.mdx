# `changeThreshold`
Defined in [OwnerManager.sol](https://github.com/safe-global/safe-smart-account/blob/main/contracts/base/OwnerManager.sol)

## Overview

```solidity
function changeThreshold(uint256 _threshold) external

```

Changes the threshold of the Safe to `_threshold`.

*This can only be done via a Safe transaction.*

This function emits the [`ChangedThreshold`](../owner-events/changedThreshold.md) event.

## Parameters

### `_threshold`
- **Type:** `uint256`

New threshold.
