```solidity
function signedMessages(bytes32 messageHash) external view returns (uint256)
```

Returns a `uint256` if the `messageHash` is signed by the owner.

## Parameters

### `messageHash`
- **Type:** `bytes32`

Hash of message that should be checked.

## Return Values
- **Type:** `uint256`
