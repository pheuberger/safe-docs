# `enableModule`

Enables the module `module` for the Safe.

<Callout type='warning'>This action can only be done via a Safe transaction.</Callout>

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function enableModule(address module) external;
    }

    contract Example {
        function example() ... {
            (ISafe safe).enableModule("0x...");
        }
    }
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Parameters

### `module`

- **Type:** `address`

Module to be whitelisted.

```solidity focus=2
(ISafe safe).enableModule(
    "0x..."
);
```

## Events

### `EnabledModule`

```solidity
event EnabledModule(address module);
```

Emitted when a module is enabled for the Safe.
